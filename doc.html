    Plugin de carrousel

    Roundabout est un plugin javascript de carrousel qui permet de faire défiler des médias sous forme de listes d'éléments html, images, textes ou autre. Sa particularité est qu'il n'est que pur javascript, et a été conçut à la base pour remplacer le plugin Cycle, suite à la suppression de la bibliothéque jQuery dont il étais dépendant. Cependant, dans un souci d'optimisation, le plugin n'a conservé que les fonctionnalités nécessaires pour le fonctionnement des widget déployés dans le cadre du programme Affinitad

    Compatibilitée.

    Utilisable et testé à partir d'ie7, firefox 23.0.1, Opéra 12.16, Safari 4.0.2 et Chrome 28.0.1500.95 m

    Le déployer.

    Exemple simple :

    Le HTML :

    <div id="example1">
  <div class="example1">
	<img width="100" height="29" alt="" src="i/affinitad.png" />
  </div>
  <div class="example1">
	<img width="100" height="21" alt="" src="i/twenga.png" />
  </div>
  <div class="example1">
	<img width="100" height="45" alt="" src="i/tangou.png" />
  </div>
  <div class="example1">
	<img width="100" height="16" alt="" src="i/showcase.png" />
  </div>
  <div class="example1">
	<img width="100" height="36" alt="" src="i/adexmachina.png" />
  </div>
</div>

    Le javascript :

    allnodeExample1 = ra.getElementsByClassName(document.body,"example1");
ra.carouselize({noeud:allnodeExample1});

    Le CSS :

    #example1{width:100px;height:45px;overflow:hidden;border:1px solid #555;}
.example1{width:100px;}

    Principes :

    • Pour la structure HTML, Avoir un div conteneur (#example1) qui contienne une liste d'éléments enfants (.example1). Ces éléments enfants peuvent contenir tout et n'importe quoi.
    • En javascript, après avoir appelé le script (<script src="votrechemin/carouselHorizontal.js"></script>) le plugin est contenu dans l'objet "ra" qui contient plusieurs méthodes, notamment la méthode getElementsByClassName qui vous permettre de récupérer et de stocker la liste d'éléments que vous souhaiterez voir défiler. Il vous suffit de passer cette liste de noeud au paramètre "noeud".
    • En CSS, il faut affecter une valeur fixe pour la largeur et la hauteur de l'élément parent, qui sera la "lucarne" dans laquelle défilera vos élèments. idéalement, chaque noeud enfant doit au moins avoir la même largeur, au risque de voir ceux-ci débordé dans la lucarne.

    Options et leur Paramètres

    Il y a suffisament de paramètres pour gérer des interactions avec un compteur, des miniatures, des flêches ou différentes actions bien déterminées.

    • noeud Un objet listant les noeuds DOM qui ont vocation à défiler.
    • thumbnail Un objet listant les noeuds DOM qui serviront de miniatures. Une classe "activeSlide" est automatiquement attribué à la miniature associée à l'élément du slide ayant le focus.
    • thumbHover Valeur booléenne qui prendra en compte l'event mouseover sur les miniatures. le paramètre thumbnail doit exister pour que ce paramètre fonctionne.
    • endRotation Temps en millisecondes au bout duquel la rotation stop. Par défaut, l'animation ne s'arrête jamais.
    • compteur Chaine représentant l'attribut ID direct ou s'affichera le numéro de la slide active.
    • btNext Chaîne représentant l'attribut ID du noeud DOM auquel sera associé l'action "slide suivante".
    • btPrev Chaîne représentant l'attribut ID du noeud DOM auquel sera associé l'action "slide précédente".
    • timer Le temps en millisecondes entre deux début de défilement. Par défaut, à 2000.
    • interactStopAll Valeur booléenne pour indiquer si oui ou non on stop le carousel lors d'un clique sur un des éléments d'interface de celui-ci. Pour les flêches ou les miniatures.
    • AnimFrequency Temps en millisecondes qui est la fréquence utilisée pour l'animation de rotation des éléments. Par défaut, initialisé à 30.
    • AnimSplitdistance Valeur numérique indiquant la distance parcourue en pixel durant l'animation de rotation entre chaque intervalles de temps dans le paramètre potentiellement définit au dessus. Par défaut, initialisé à 1 pixel. Cette valeur s'incrémente automatiquement au cours de l'animation, d'un pixel supplémentaire.

    Fonctions supplémentaires

    Afin de faire fonctionner le carrousel, des fonctions mimiquant certaines natives très pratique de jquery ont été créées. Vous pouvez donc automatiquement en disposer partout ou vous inclurez le script de roundabout. 

    • ra.getElementsByClassName = function(wrapNode, classname)
      • wrapNode : un objet DOM à partir duquel la recherche va s'effectuer (pour éviter de parser tout un document si besoin)
      • classname : la classe recherché
      • action : renvoie un objet contenant les noeuds DOM
    • ra.setCssAttribute = function(domNode, keyValue)
      • domNode : le noeud dont vous souhaitez éditer les CSS
      • keyValue : la ou les couples de clés/valeurs des attributs css à setter, sous la forme d'une chaîne de caractères. Exemple : "position:absolute;left:25px;"
      • action : va attribuer les CSS au noeud ciblé, sans pour autant affecter les existants ou surcharger.
    • ra.addClass = function(nodeToCheck, cl)
      • nodeToCheck : l'objet noeud sur lequel vous souhaitez ajouter une classe.
      • cl : Une chaîne de caractère contenant le nom de la classe.
      • action : va ajouter la classe souhaitée sur le noeud DOM
    • ra.removeClass = function(nodeToCheck, cl)
      • nodeToCheck : l'objet noeud sur lequel vous souhaitez retirer une classe existante.
      • cl : Une chaîne de caractère contenant le nom de la classe à retirer.
      • action : va retirer la classe ciblée du noeud DOM
    • ra.attachAnEvent = function(evName,NId,fnName)
      • evName : Une chaîne de caractère contenant le nom de l'événement DOM. Ex : "mouseover". Attention, la compatibilité dépendant du support de DOM par les navigateurs n'est pas assurée, excepté pour l'événement click. En clair, seul pour la différences "click" / "onclick" a été gérée, pour les autres, il faudrait améliorer le plugin.
      • NId : L'objet du noeud DOM associé à l'événement.
      • fnName : Une fonction qui va être associée à l'événément et aux noeud DOM passé précédement en paramètre.
      • action : Va permettre de déclencher la fonction lorsque l'utilisateur interagira sur le noeud DOM souhaité
    • ra.detachAnEvent = function(evName,NId,fnName)
      • evName : Une chaîne de caractère contenant le nom de l'événement DOM duquel dissocier une fonction précédemment affectée via la fonction vue juste au dessus.
      • NId : L'objet du noeud DOM ciblé.
      • fnName : L'occurence de la fonction a dissocier du noeud DOM.
      • action : Va permettre de retirer la fonction de l'événement DOM. Permet notamment d'éviter l'itération d'événements sur un noeud DOM qui améne le script inutilisable.

    Exemples

    Avec boutons permettant de faire précédent et suivant

    allnodeExample2 = ra.getElementsByClassName(document.body,"example2");
ra.carouselize({
	noeud:allnodeExample2,
	btNext:"arrowNext",
	btPrev:"arrowPrev"
});

    Carousel avec temps d'exposition des slides beaucoup plus long

    allnodeExample4 = ra.getElementsByClassName(document.body,"example4");
ra.carouselize({
	noeud:allnodeExample4,
	timer:4000
});

    Avec miniatures (qui réagissent par défaut au clique)

    allnodeExample3 = ra.getElementsByClassName(document.body,"example3");
thumbnail3 = ra.getElementsByClassName(document.body,"thumbs");
ra.carouselize({
	noeud:allnodeExample3,
	thumbnail:thumbnail3
});

    Avec miniatures (sous formes de simple liens texte) qui réagissent au rollover

    Affinitad Twenga Twenga Chine Showcase Adexmachina 
    allnodeExample5 = ra.getElementsByClassName(document.body,"example5");
thumbnail4 = ra.getElementsByClassName(document.body,"thumbs2");
ra.carouselize({
	noeud:allnodeExample5,
	thumbnail:thumbnail4,
	thumbHover:true
});

    Avec un compte à rebours qui stop le carousel au bout de 14 secondes

    allnodeExample6 = ra.getElementsByClassName(document.body,"example6");
ra.carouselize({
	noeud:allnodeExample6,
	endRotation: 14000
});

    Lors du clique sur les miniatures ou sur les flêches, on stop la continuité de l'animation du carousel.

    allnodeExample7 = ra.getElementsByClassName(document.body,"example7");
thumbnail1 = ra.getElementsByClassName(document.body,"thumbs1");
ra.carouselize({
	noeud:allnodeExample7,
	thumbnail:thumbnail1,
	btNext:"arrowNext0",
	btPrev:"arrowPrev0",
	interactStopAll: true
});

    Exemple d'utilisation des paramètres AnimSplitdistance et AnimFrequency.

    allnodeExample8 = ra.getElementsByClassName(document.body,"example8");
ra.carouselize({
	noeud:allnodeExample8,
	AnimFrequency:100,
	AnimSplitdistance:2
});

    Exemple d'utilisation du paramètre compteur, qui permet d'afficher le numéro de la slide courante.

    1
    /5 
    allnodeExample9 = ra.getElementsByClassName(document.body,"example9");
thumbnail3 = ra.getElementsByClassName(document.body,"thumbs3");
ra.carouselize({
	noeud:allnodeExample9,
	thumbnail:thumbnail3,
	btNext:"arrowNext1",
	btPrev:"arrowPrev1",
	compteur:"counterIndicator"
});
